# Categorías de Productos - Plomería García

Versión: 1.0

---

# Objetivo

Este documento define las categorías comerciales utilizadas por Plomería García.

El agente de IA utilizará esta información para:

* Clasificar productos.
* Facilitar búsquedas.
* Recomendar productos relacionados.
* Realizar ventas cruzadas.
* Generar estadísticas por categoría.
* Comprender mejor el catálogo completo de la empresa.

---

# Regla General

Cada producto deberá pertenecer al menos a una categoría principal.

Si un producto puede pertenecer a varias categorías, siempre deberá existir una categoría dominante.

Ejemplo:

Monomando para lavabo

Categoría principal:

Grifería

Categoría secundaria:

Baños

# Prioridad Comercial

El agente deberá conocer cuáles categorías tienen mayor prioridad comercial.

La prioridad podrá cambiar dependiendo de:

* Temporada
* Promociones
* Margen
* Inventario
* Objetivos de ventas

---

# Estadísticas

Cada categoría podrá almacenar:

* Número de productos
* Productos activos
* Productos descontinuados
* Ventas mensuales
* Ticket promedio
* Margen promedio
* Clientes frecuentes

---

# Uso por la IA

El agente utilizará estas categorías para:

* Recomendar productos.
* Hacer ventas cruzadas.
* Buscar productos.
* Analizar ventas.
* Detectar oportunidades.
* Sugerir promociones.
* Ayudar al vendedor durante la atención al cliente.

---

# Escalabilidad

Las categorías deberán poder crecer sin modificar la arquitectura del sistema.

Será posible agregar nuevas categorías y subcategorías en cualquier momento.

---

# Nota Importante

Las categorías definidas en este documento representan la estructura lógica del catálogo.

La información oficial de productos siempre provendrá del sistema Intelisis y del catálogo vigente de Plomería García.

Si existe una diferencia entre este documento y el catálogo oficial, prevalecerá la información del catálogo oficial.
