# PG Lista 3 AI

## Documento Maestro del Proyecto

Versión: 1.0

---

# Misión

Crear la plataforma más inteligente para ayudar a los asesores comerciales de Plomería García a conseguir nuevos clientes Lista 3, administrar su cartera, aumentar las ventas y mejorar el seguimiento comercial utilizando Inteligencia Artificial.

El objetivo del sistema no es únicamente registrar clientes.

El objetivo es ayudar al vendedor a vender más.

---

# Problema actual

Actualmente gran parte del trabajo comercial depende de la memoria del vendedor.

Se pierden oportunidades porque:

* No existe seguimiento constante.
* Se olvidan llamadas.
* Se olvidan cotizaciones.
* No se identifican clientes inactivos.
* No existe una agenda inteligente.
* No hay prioridades automáticas.
* No existe análisis de oportunidades.
* No hay una vista completa de cada cliente.

Esto provoca pérdida de ventas.

---

# Solución

Construir un asistente inteligente que funcione como un copiloto del vendedor.

El sistema deberá ayudar a tomar decisiones todos los días.

Ejemplos:

• ¿A quién debo llamar hoy?

• ¿Qué cliente tiene una cotización pendiente?

• ¿Quién dejó de comprar?

• ¿Qué constructora lleva varios meses sin comprar?

• ¿Qué cliente tiene potencial de crecer?

• ¿Qué seguimiento debo realizar?

La aplicación responderá estas preguntas automáticamente.

# Objetivo de la Inteligencia Artificial

La IA deberá comportarse como un gerente comercial.

No deberá limitarse a responder preguntas.

También deberá proponer acciones.

Ejemplos:

"Este cliente tiene 45 días sin comprar."

"Sería buena idea llamarlo."

"Este cliente normalmente compra impermeabilizante en estas fechas."

"Esta cotización lleva una semana sin seguimiento."

"La probabilidad de compra es alta."

---

# Reglas generales

La IA siempre utilizará datos reales.

La IA explicará por qué hace cada recomendación.

La IA deberá ser transparente.
