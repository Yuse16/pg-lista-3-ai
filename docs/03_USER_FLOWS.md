# Flujos de Usuario

Proyecto: PG Lista 3 AI

Versión: 1.0

---

# Objetivo

Este documento describe cómo interactúan los usuarios con la aplicación.

Todos los flujos deberán ser simples, rápidos y orientados a incrementar las ventas.

La prioridad es reducir el tiempo administrativo del vendedor y aumentar el tiempo dedicado a atender clientes.

---

# Flujo 01 - Inicio de sesión

Usuario abre la aplicación.

↓

Introduce sus credenciales.

↓

El sistema valida el acceso.

↓

Carga la información correspondiente a su sucursal y permisos.

↓

Muestra el Dashboard Principal.

---

# Flujo 02 - Dashboard diario

Al iniciar sesión el usuario verá un resumen de su día.

Información mostrada:

* Seguimientos pendientes.
* Clientes prioritarios.
* Cotizaciones pendientes.
* Nuevos prospectos.
* Indicadores de ventas.
* Actividad reciente.
* Recomendaciones de la IA.

Objetivo:

Que el usuario sepa inmediatamente qué debe hacer.

# Flujo especial - Asistente Inteligente

Este será el flujo más importante del proyecto.

Al abrir la aplicación la IA analizará automáticamente:

* Clientes sin seguimiento.
* Clientes inactivos.
* Cotizaciones pendientes.
* Próximas tareas.
* Compras recientes.
* Actividad del usuario.

Con base en ello generará un plan de trabajo personalizado.

Ejemplo:

Hoy se recomienda:

1. Llamar a Constructora Alfa.

Hace 18 días recibió una cotización.

2. Contactar a Arquitectura Delta.

No existe seguimiento desde hace 12 días.

3. Revisar al cliente Industrial Norte.

Anteriormente realizaba compras mensuales y lleva 45 días sin comprar.

---

# Reglas de experiencia de usuario

Todas las acciones importantes deberán completarse en menos de 3 clics.

El usuario nunca deberá navegar por múltiples pantallas para registrar información básica.

La aplicación deberá responder rápidamente.

La IA nunca deberá interrumpir el trabajo del usuario.

Las recomendaciones deberán ser útiles y fáciles de entender.

---

# Principio del flujo de usuario

Cada vez que el vendedor abra la aplicación deberá salir con una respuesta clara a la pregunta:

**¿Qué debo hacer hoy para vender más?**
